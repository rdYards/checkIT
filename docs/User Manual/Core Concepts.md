Before diving in, it is helpful to understand the two main components of the app:

*   **The Ledger:** Think of a Ledger as a secure "vault" or "digital notebook." Each ledger is an independent, encrypted file. You can have multiple ledgers (e.g., one for "Work," one for "Personal," one for "Finance").
*    **The Entry:** Inside every ledger, you record "Entries." An entry consists of a **Genre** (a category or tag like *add*, *remove*, or *modify*), a **Timestamp**, and the **Data** (your actual note or record).
