If you want to create a duplicate of an existing ledger (for example, to use as a template), use the Clone feature.
1.  Open the ledger you wish to copy.
2.  Click the **Settings Icon** next to the ledger name in the sidebar, or press **`Ctrl + Shift + C`**.
3.  Select **Clone**.
4.  Enter a **New Title** and a **New Password** for the duplicate.
5.  CheckIT will create a new, separate encrypted file containing all the data from the original.
