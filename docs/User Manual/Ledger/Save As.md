Use this to create a new `.sl` file in a different location, or to "export" a copy of your current ledger.
1.  Click the **Settings Menu** in the sidebar.
2.  Select **Save As**, or press **`Ctrl + Shift + S`**.
3.  A file browser will open. Choose your destination folder and name your new file.
4.  A prompt will appear asking you to **Enter Password**. This is the password used when creating the Ledger and will be used to encrypt the new file you just created.
