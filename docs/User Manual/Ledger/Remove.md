To remove a ledger from your active list:
1.  Open the ledger you want to delete.
2.  Click the **Settings Menu** in the sidebar, or press **`Ctrl + Delete`**.
3.  Select **Remove**.
4.  A confirmation dialog will ask if you want to save changes first. 
    *   *Note: Removing a ledger from the app list does not delete the physical file on your hard drive unless you have manually deleted the file from your computer.*
