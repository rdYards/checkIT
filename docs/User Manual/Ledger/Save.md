CheckIT does **NOT** auto save your ledger. This is a planned future update.

Use this when you have imported a ledger and want to save your latest changes back to the original file on your and hard drive.
*Note: If you are working on a brand-new ledger that has never been saved to your computer, selecting "Save" will automatically redirect you to the "Save As" process.*
1.  Click the Settings Menu (the icon next to the ledger name) in the sidebar, or press `Ctrl + S`.
2.  Select Save.
3.  A dialog will appear asking you to Enter Password. Enter the password for that ledger, the same password used to create the Ledger, to authorize the overwrite.
